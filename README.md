# README 2DBEM GUI
透過二維邊界元素法（BEM）求解２維翼型的升力係數（CL）與壓力分佈（-CP）

![image]([https://fakeimg.pl/500/](https://github.com/KWGHG/2DBEM_GUI/blob/master/2DBEM_GUI.jpg))

## 用法
可參考example資料夾內的範例  
:book:example資料夾內容
+ `2DBEM.exe`  GUI持行檔
+ `/input` 輸入檔資料夾，內有常見翼型幾何

## 安裝
此專案由python開發，需要套件如`requirements.txt`所示。  
套件：  
+ `PyQt5`
+ `matplotlib`
+ `numpy`
+ `scipy`
+ `pyinstaller`
